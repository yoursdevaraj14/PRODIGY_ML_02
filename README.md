# 🛍️ Customer Segmentation using K-Means Clustering

## 📌 Overview

This project uses the K-Means Clustering algorithm to group retail store customers based on their annual income and spending score. Customer segmentation helps businesses understand customer behavior and create targeted marketing strategies.

## 📂 Dataset

Dataset: Customer Segmentation Tutorial in Python (Kaggle)

File used:
- Mall_Customers.csv

Features Used:
- Annual Income (k$)
- Spending Score (1-100)

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

## 🤖 Algorithm

- K-Means Clustering

## 📊 Workflow

- Load the dataset
- Select relevant features
- Find the optimal number of clusters using the Elbow Method
- Apply K-Means Clustering
- Visualize customer segments and cluster centroids

## 📈 Output

- Elbow Method graph
- Customer Segmentation scatter plot
- Cluster centroids visualization

## ▶️ How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the program:

```bash
python customer_segmentation.py
```
